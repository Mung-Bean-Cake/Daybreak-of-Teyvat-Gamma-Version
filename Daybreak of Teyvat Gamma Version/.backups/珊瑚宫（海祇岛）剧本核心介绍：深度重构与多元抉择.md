# 珊瑚宫（海祇岛）剧本核心介绍

本剧本基于《原神》稻妻篇海祇岛珊瑚宫设定深度重构，以**珊瑚宫心海**为核心领袖，围绕「海祇大御神奥罗巴斯的遗志」「眼狩令与反抗军起义」「鸣神幕府的世仇与对抗」「渊下宫白夜国的遗产」「提瓦特诸国的外交博弈」五大核心脉络展开。

剧本特色：

1. **深度派系政治**：还原海祇岛内部主战派、主和派、保守派、革新派的路线斗争，不同抉择将彻底改变国家走向

2. **双线史诗叙事**：既有反抗幕府压迫的战场史诗，也有探索渊下宫、揭开奥罗巴斯牺牲真相的深海秘闻

3. **多结局自由抉择**：包含「推翻幕府重塑稻妻」「与鸣神达成永恒和解」「海祇独立建国」「提瓦特海洋霸权」四大终极路线

4. **高还原度机制**：贴合原神原作设定的「海祇军战术体系」「巫女智囊团」「龙蜥研究」「邪眼事件」等核心玩法，兼顾趣味性与原作沉浸感
